
## ⚡ A Modern iOS-Style Professional Portfolio

### 🧠 Overview
This is a high-performance, mobile-first portfolio engineered with an iOS-inspired interface. It delivers a smooth, app-like experience while presenting projects, certifications, and technical capabilities in a structured and visually refined format.  

This project functions as a **professional digital identity**, not just a portfolio.

---

### ✨ Key Highlights
- 📱 iOS-style UI with glassmorphism dock  
- ⚡ Smooth animations and interaction feedback  
- 🔗 Dynamic GitHub repository integration  
- 🧩 Modular and scalable UI structure  
- 📜 Certifications and achievements display  
- 🎯 Clean typography with Noto Sans  
- 🚫 No frameworks — pure Vanilla JavaScript  

---

### 🛠️ Tech Stack
- **HTML5** — Structure  
- **CSS3** — Styling (modern layout + variables)  
- **JavaScript** — Logic (Vanilla)  
- **GitHub REST API** — Dynamic data  

---

### 🧱 Architecture
- 👤 **Hero Section** — Identity and branding  
- 🔗 **Action Row** — External links  
- 📄 **About Section** — Profile summary  
- 🏆 **Achievements** — Academic highlights  
- 🧠 **Capabilities** — Skill categorization  
- 💻 **Projects** — Dynamic + manual rendering  
- 📜 **Certifications** — Structured listing  
- 📦 **Modal System** — Extended content view  
- 🧭 **Floating Dock** — iOS-style navigation  

---

### 🔄 Dynamic GitHub Integration
Projects are fetched in real-time from:

`https://api.github.com/users/psvineet/repos`

#### ⚙️ Behavior
- Merges GitHub repositories with manual entries  
- Filters unnecessary repositories  
- Keeps portfolio automatically updated

### 🚀 Deployment
- 🌐 GitHub Pages  
- ▲ Vercel  
- ⚡ Netlify  

#### GitHub Pages Setup
1. Push repository  
2. Go to **Settings → Pages**  
3. Select **main branch**  
4. Deploy from root
